If using Git with a local copy of RStudio in Windows and Git Bash
==============================================================
You will need to tell R-Studio where the "git" executable is located on your system.
In the RStudio menu go to Tools->Global Options then select Git/SVN.
Enter the path where git is located. On my Windows 7 system, it was installed here:
C:/Program Files (x86)/Git/bin/git.exe.
Enter the correct path for the git.exe and select OK
